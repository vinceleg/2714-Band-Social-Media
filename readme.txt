Band Social Media

Application description: 
The following application allows users to add a band or an event.
To add a band, you must include the band's name, number of members, and music genre.
To add an event, you must include the event's name, location, and date.
Created bands and events will be stored in a MySQL database.
You will be able to see the list of created bands and users from 127.0.0.1:3000

Guide:
Create your database.
1. Enter the contents from the "band.sql" file to a query tab in MySQL workbench.

2. Open "config.js" file in a word editor and change the 'user' and 'password' fields to your MySQL database user and password.

3. From the command prompt, navigate to the folder that contains the bandsocialmedia folder. Type in "node app.js".

4. From a browser, go to 127.0.0.1:3000.

5. To add a user, go to the "Add New User" tab.
